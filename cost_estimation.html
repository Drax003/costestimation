💡 Cost Saving Tips Buy E32 LoRa modules from AliExpress in bulk — ₹350 each online vs ₹600+ local shops. Use Google Colab (free) for AI model training instead of paid GPU servers. Use Mosquitto MQTT on a free Oracle Cloud VM instead of paid AWS.
⚠️ Price Warning SIM7600 is the most expensive component. For prototype-only testing, skip it and use only IR + LoRa. Add 4G later in Phase 2.
📦 College Project Tip For a college demo, use the free Wokwi simulator instead of buying all hardware. Buy only ESP32 + OLED + buzzer (₹650 total) to demonstrate the core logic.
